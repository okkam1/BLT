Adding Client Information:

 

Onboard Client Web Page

1.      From the Main Menu Web Page, select Onboard Lead Research Study Client

2.      Select an existing family from the dropdown list (expedite search by typing in first few letters of last name) or click Add New Family.

3.      If adding a new family:

a.       Input new family information using blanks provide. The following information is required:

                                                              i.      Family Name

                                                           ii.      Date Built

b.      Click Insert Family.

c.       To enter an additional family, click Add Another.

d.      To proceed to enter a person, click Next

4.      Enter birthdate in xx/xx/xxxx form or click on associated calendar. You can click on the calendar headings to zoom in and out of dates.

5.      Continue entering data, using dropdown menus when available. The following information is required:

a.      Name

b.     Birthdate

c.      Gender

d.     Language

e.      Ethnicity

6.      Click Save and Continue.

7.      To enter an additional person, click Add Another.

8.      To proceed to enter Questionnaire data, click Next

 

NOTE: Questionnaire and Blood Lead Test results will be entered for the most recent client entered.

 

Questionnaire Web Page

1.      Continue entering in data on Questionnaire web page. All dates should be entered in xx/xx/xxxx format.

2.      Click Save and Continue when complete.

3.      To enter an additional questionnaire for the same person, click Add Another

4.      To proceed and enter Blood Test Results, click Next

 

Blood Lead Test Results Page

5.      Enter Sample Date and Lab Date using xx/xx/xxxx format or click on associated calendar. You can click on the calendar headings to zoom in and out of dates.

6.      Continue entering data into the rest of the fields. The following information is required:

a.      Source of Results

NOTE: A blood lead value of -3.3 indicates a reading of less than 3.3 from the LeadCare II analyzer.

7.      To enter additional Blood Lead Test results, click Add Another.

8.      To finish entering data for the client click Save and Done.

Edit Existing Information

1.      From the Main Menu web page, select Edit Existing Lead Research Study Client

2.      Select the appropriate Item to edit:

a.       Edit Existing Client Information

b.      Edit Existing Family Information

c.       Edit Existing Client Blood Level Information

d.      Edit Existing Client Questionnaire Information

 

Edit Existing Client Information

1.      Select the family from the Family drop down list.

2.      Select the individual from the Family Members dropdown list.

3.      Existing information will automatically populate the current fields, except for the notes fields.

a.       Historic notes can be reviewed HOW?

b.      New notes can be entered, and will be added to the clients historic notes.

4.      Update any information that needs to be changed.

5.      Click Update.

 

Edit Existing Family Information

1.      Select the family from the Family drop down list.

2.      Existing information will automatically populate the current fields, except for the notes fields.

a.       Historic notes can be reviewed HOW?

b.      New notes can be entered, and will be added to the familys historic notes.

3.      To add a new address, select Add New Address

a.       This will clear all existing address information and enter a completely new address into the database.

4.      To edit an existing address, select Edit Existing Address

a.       This will change the data for the existing address in the database.

5.      Update any additional information that needs to be changed.

6.      Click Update Family Information.

7.      Click Close on the confirmation pop up.

 

Edit Existing Client Questionnaire Information

1.      Select the family from the Family drop down list.

2.      Select the individual from the Family Members dropdown list.

3.      Existing information will automatically populate the table of historic questionnaire data.

4.      To edit information for a specific questionnaire, click Edit next to the appropriate questionnaire.

5.      Update any information that needs to be changed.

a.       To commit the updates, click Update.

b.      To discard the updates, click Cancel.

6.      To insert a new questionnaire for the current client, click Insert New Questionnaire

 

Edit Existing Client Blood Level Information

1.      Select the family from the Family drop down list.

2.      Select the individual from the Family Members dropdown list.

3.      Existing information will automatically populate the table of historic Blood Lead Tests data.

4.      To edit information for a specific blood test, click Edit next to the appropriate test results id.

a.       Update any information that needs to be changed.

                                                               i.      To commit the updates, click Update.

                                                             ii.      To discard the updates, click Cancel.

5.      To delete data from a blood test, click Delete next to the appropriate test results id.

a.       Warning: this will delete the test results without any request for confirmation. Make sure you intend to delete the results prior to clicking the delete. Once the data has been deleted, it is a non-trivial process to recover.

6.      To insert new blood lead level test results for the current client, click Insert New Results

Reports:

 

Reports are available for all data entered for Study Clients.

 

Navigating Report Pages

1.      Multiple pages of the reports can be viewed by clicking the arrow located at the top of each report.

2.      To jump to a specific report page, enter the page number into the box at the top of the report and then press the right arrow.

3.      To jump to the first page of a report, press the left arrow with the vertical bar next to it

4.      To jump to the last page of a report, press the right arrow with the vertical bar next to it

 

Accessing Reports:

1.      From the Main Menu, click on View Reports for Lead Research Study

2.      The following reports are available:

 

1.      Summary Report: A count of clients, new clients, blood tests, blood lead levels between 5 and 9.9, elevated blood levels (10ug/dl and above), pregnant women, nursing mothers, nursing infants, adults, and home visits and soil testing conducted during the selected date range*. The default date range starts today and ends 1 month in the past.

2.      Client Follow Up List: A list of clients that should be followed up with during the specified date range*. The default date range starts today and ends 1 month in the future.

3.      List of Clients by Created Date: A list of clients that were added to the Blood Lead Study during the specified date range*. The default date range starts today and ends 1 month in the past.

4.      List of Clients by Modified Date: A list of clients that were modified during the specified date range*. The default date range starts today and ends 1 month in the past.

5.     List of Clients by Blood Lead Level: A list of clients with blood lead levels in the specified range**. The default range is from -3.3*** to 100.

6.      Count of Clients by Age Group: A pie chart of clients grouped by age** according to the following ranges:

a.       0

b.      01 03

c.       04 06

d.      07 17

e.       18 and over

 

*NOTE: Date ranges include data from 12:00 am on the start date to 11:59pm on the end date.

**NOTE: Blood lead level ranges include data with blood lead values equal to the minimum and maximum as well as all values between.

***NOTE: A blood lead value of -3.3 indicates a reading of less than 3.3 from the LeadCare II analyzer.

****NOTE: age ranges include individuals with ages equal to the minimum and maximum as well as all values between.

 

Searching for Specific Data within Any Report:

1.      Click on report you wish to search.

2.      Enter search term (i.e. last name) into search box located at the top of the report.

3.      Click Find.

4.      Search term should be highlighted.

5.      To move through the search results, click next until you have found the desired highlighted data.

 

Exporting Report Data:

1.      Click on report you wish to export.

2.      Click floppy disk picture located at the top of the report.

3.      Choose the format for the exported data:

a.       Excel

b.      PDF

c.       Word
